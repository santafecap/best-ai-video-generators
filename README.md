# Best AI Video Generators
Curated list of the best AI video generation models and services. Please send a PR to add yours today!

### Models

- [Sora 2](https://openai.com) OpenAI's text-to-video model that can generate high-quality, realistic videos up to 60 seconds long from text descriptions. It understands physics and motion and can create complex scenes with multiple characters and specific movements.
- [Minimax](https://minimax.io) A Chinese AI video model that has gained attention for generating high-quality, realistic videos with motion direction. It's particularly noted for its ability to handle complex physics and realistic motion.
- [Runway Gen-3 Alpha](https://runwayml.com) The latest generation from Runway, offering high-fidelity, controllable video generation with improved consistency and temporal coherence. It's widely used by creative professionals and supports various video editing workflows.


### Services

- [Republiclabs.ai](https://www.republiclabs.ai/) - Republiclabs.ai, a multi-model platform that aggregates and curates the best SOTA AI video models all in a single place for users to generate content quickly. 
- [DreamStudio](https://beta.dreamstudio.ai/) - DreamStudio is an easy-to-use interface for creating vidoes using the Stable Video Diffusion video generation model.
